# 🎨 CSS Full Course — Level 3

![CSS Level 3 Banner](https://copilot.microsoft.com/th/id/BCO.11e3e234-d4be-4a88-8aa5-a435ada62b9d.png)

Welcome to **CSS Full Course — Level 3**!  
This repository is designed for learners who want to advance their knowledge of **Cascading Style Sheets (CSS)**. It focuses on practical concepts like **positioning** and **relative units**, which are essential for building responsive and well-structured web pages.

---

## 📚 Course Overview

By the end of Level 3, you will be able to:

- Understand and apply **CSS positioning** (static, relative, absolute, fixed)
- Use **relative units** like `em`, `rem`, `%`, and `vh/vw` for scalable design
- Create layouts that adapt to different screen sizes
- Combine positioning and units to build **responsive web components**
